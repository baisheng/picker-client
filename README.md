# 一款基于 Quasar + Graphql + hasura 实现的资源站产品

## 安装 Install the dependencies
```bash
yarn
```

### 开发 Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
docker-compose up -d
quasar dev
```

### Lint the files
```bash
yarn run lint
```

### 发布 Build the app for production
```bash
docker-compose up -d
quasar build
```
